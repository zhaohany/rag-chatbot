# Sample Test Queries (Internal IT Support)

Use these queries after running `POST /ingest` to sanity-check retrieval quality.

## Password and Access

- How does IT Support verify user identity before password reset?
- What are the exact steps for account unlock?
- What is the SLA target for password reset and MFA recovery?
- When should account unlock be escalated to Security?

## VPN and Connectivity

- What should I check first when VPN cannot connect?
- When should VPN issues be escalated to the Network team?
- What troubleshooting steps are required before escalation?

## Email and Endpoint Support

- What should IT Support do if Outlook keeps asking for login?
- What is the standard new laptop setup checklist?
- What is the policy for local admin rights?

## Printer and Device Issues

- What are the standard steps if printer queue is stuck?
- How do we remap a missing network printer?
- When should a slow laptop be reimaged?

## Realistic User Questions

- I forgot my password and changed my phone, so MFA is broken. What is the recovery process?
- My VPN fails with timeout but internet is working. What diagnostics should support run before escalation?
- My Outlook is not syncing and I have repeated auth prompts. What should be checked in order?
