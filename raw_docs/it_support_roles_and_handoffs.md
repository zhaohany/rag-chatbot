# Device Setup and Software Installation SOP

## New Laptop Setup (First Day)
1. Verify user identity and asset tag.
2. Join device to company domain.
3. Install security tools: endpoint protection and disk encryption agent.
4. Install standard apps: Office suite, browser, VPN client, collaboration tools.
5. Confirm email, calendar, and shared drive access.

## Standard Software Request
User submits software request ticket with business reason and manager approval.
IT Support checks license availability and compatibility with endpoint policy.
Approved software should be installed within 1 business day.

## Admin Rights Policy
Local admin rights are not granted by default.
Temporary elevation must include ticket reference, duration, and approval owner.

## Reimage Procedure
Backup user files to approved cloud or shared storage.
Reimage the device with current baseline image.
Restore approved files and validate business-critical apps.

## Final Validation Checklist
User can sign in with SSO.
VPN connects successfully.
Printer and shared folders are accessible.
Security tools report healthy status.
