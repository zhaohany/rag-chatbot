# Common IT Troubleshooting Playbook

## VPN Not Connecting
Symptoms: connection timeout, authentication failed, or no route to internal services.
Checklist:
1. Confirm user credentials and MFA status.
2. Verify client version and certificate validity.
3. Test network from alternate Wi-Fi or hotspot.
4. Flush DNS cache and restart VPN client.
5. Escalate to Network team if gateway health is degraded.

## Outlook or Email Sync Issues
Symptoms: cannot send/receive, mailbox not updating, repeated login prompt.
Checklist:
1. Verify mailbox quota and service health advisory.
2. Re-authenticate account and clear cached credentials.
3. Rebuild local profile if sync still fails.
4. Collect error code and timestamp before escalation.

## Printer Not Working
Symptoms: printer offline, queue stuck, missing printer mapping.
Checklist:
1. Confirm device and printer are on same network segment.
2. Clear print queue and restart print spooler service.
3. Re-add printer from approved print server.
4. Reinstall signed printer driver package.

## Slow Laptop Performance
Symptoms: startup delay, high CPU, frequent freezing.
Checklist:
1. Check available disk space and memory usage.
2. Remove unauthorized startup apps.
3. Run endpoint health scan and patch update.
4. If unresolved, schedule backup and reimage.
